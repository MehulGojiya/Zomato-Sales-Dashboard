# Zomato-Sales-Dashboard
An interactive Power BI dashboard analyzing Zomato data to uncover key insights on restaurant performance, customer preferences, ratings, and order trends. This project includes data cleaning, transformation, and visualization to support data-driven decision-making using KPIs, charts, and filters.
